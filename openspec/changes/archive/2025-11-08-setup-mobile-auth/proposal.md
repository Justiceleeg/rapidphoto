# Change: Setup Mobile Frontend - Auth

## Why
Initialize the Expo mobile application with Expo Router, configure it for the monorepo, setup React Query and state management, implement authentication client integration, and create login/register screens. This enables users to register and login through the mobile interface.

## What Changes
- Initialize Expo project in `apps/mobile`
- Configure for monorepo with workspace dependencies
- Setup Expo Router for navigation
- Configure React Query providers
- Implement authentication client and hooks
- Create login and register screens

## Impact
- Affected specs: mobile (new capability)
- Affected code: `apps/mobile/` directory structure, authentication screens, and API client package
